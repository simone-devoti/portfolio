Frequently asked questions

Why does this app exist?

Its purpose is just to showcase some of my skill

What does it do?

Enables you to create and manage your to-dos. It follows the principles of the Eisenhower matrix. If you organize your tasks in this matrix you will be more productive

How was it built?

It uses Blazor WASM. A dotnet framework that can be used to build client rendered web applications using C# and compiling it to webassembly code. I enabled the PWA support, so you can download the app into your phone, tablet or computer.

Yes you can! The source code is available on my Github profile
